Project Name: Travellite

Description:
This project is a responsive tourist website designed to promote travel destinations and provide useful information for visitors. It includes interactive elements such as image galleries, maps, and contact forms built with HTML, CSS, and JavaScript.

Responsive design for desktop and mobile

Technologies Used:

HTML5

CSS3

JavaScript
